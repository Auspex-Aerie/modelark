# Support

ModelArk is pre-1.0 community software and has no paid support or response-time guarantee.

- Usage questions and reproducible bugs: [GitHub Issues](https://github.com/Auspex-Aerie/modelark/issues)
- Security vulnerabilities: follow [`SECURITY.md`](SECURITY.md), not a public issue
- Data-loss or integrity bugs: prefix the issue title **`[P1]`** and include the command, commit,
  relevant logs, and whether the affected drive remains mounted

Please search existing issues first and remove tokens, private host details, drive serials, UUIDs,
and local paths from logs before posting.
